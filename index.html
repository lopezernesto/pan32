Tecnologías que manejo

Tecnologías frontend:

HTML5: Avanzado

lenguaje de maquetado de páginas web. Amplio conocimiento en el uso del lenguaje en cuestión.

CSS3: Medio

Lenguaje de estilos. son los que hacen bonita a la página. Conocimiento medio, sin contar las animaciones.

JS: Medio

Lenguaje de script de web. avanzado en eventos y funcionalidades varias. Frameworks no.

Boostrap: basico

herramienta gratuita para el desarrollo de HTML, CSS y JS. paginas respocive, que se pueden ver en varios dispositivos

Tecnologías backend:

PHP: Medio

Lenguaje de servidor. controla las consultas, procesos que nos piden los usuarios. Importante para gestionar bases de datos, etc.

MySQL: Basico

lenguaje de base de datos. Importante para gestionar los datos, que almacenamos en diversas aplicaciones.

CMS (content management system o sistemas de gestión de contenidos):

Wordpress: Basico-Medio

Gestor de contenidos. Herramienta para crear paginas web de todo tipo.
