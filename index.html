tecnologias que manejo

tecnologias frontend:

HTML5: Avanzado

lenguaje de maquetado de paginas web. amplio conocimiento en el uso del lenguaje en cuestion.

CSS3: Medio

lenguaje de stilos. son los que hacen bonita a la pagina. conocimiento medio, sin contar las animaciones.

JS: Medio

lenguaje de script de web. avanzado en eventos y funcionalidades varias. Frameworks no.

Boostrap: basico

herramienta gratuita para el desarrollo de HTML, CSS y JS. paginas respocive, que se pueden ver en varios dispositivos

tecnologias backend:

PHP: Medio

lenguaje de servidor. controla las consultas, procesos que nos piden los usuarios. importante pare gestionar bases de datos, etc.

MySQL: Basico

lenguaje de base de datos. importante para gestionar los datos que almacenamos para diversas aplicaciones.

CMS (content management system o sistemas de gestion de contenidos):

Wordpress: basico-medio

gestor de contenidos. herramienta para crear paginas web de todo tipo.
